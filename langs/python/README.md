# Tetris Analyzer

A Tetris playing AI

## Prerequisites

- Python 3.8 or higher
- Poetry (install with `brew install poetry` on macOS)
- SDL2 libraries (install with `brew install sdl2 sdl2_image sdl2_mixer sdl2_ttf` on macOS)

## Installation

```bash
poetry install
```

## Usage

Run the Tetris game:

```bash
poetry run tetris
```

Alternative methods:

```bash
# Using Python directly
poetry run python tetris.py

# Or activate the Poetry shell first
poetry shell
python tetris.py
```
